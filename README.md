# fastqe

## FASTQ with Emoji

Given a fastq file in Illumina 1.8+/Sanger format, calculate the mean (rounded) score for each position and print a corresponding emoji!

![Example](fastqe.png)


## Dependencies

- pyemojify
- BioPython
- NumPy
